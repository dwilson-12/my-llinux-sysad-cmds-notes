Example:

- In Oracle VM you add first a Virtual Hard Disk in the storage tab -- Controller:SATA -- Add Hard Disk -- Create new disk  -- VDI --Dynamically allocated and so on..

# Become root
# Create a new partition from new disk
$ fsidk /dev/sdb
Command (m for help):	n = new partition
			p = primary
			partition number: default
			first sector: default
			last sector: default
			w = write table to disk
	
# Next is to assign file system type
$ mkfs.xfs /dev/sdb1

# Mount it to a certain mount point
$ mkdir /data
$ mount /dev/sdb1 /data

# Automatically mount it when system reboots
$ vi /etc/fstab

# add this line
/dev/sdb1	/data	xfs	defaults	0	0

# How to unmount
$ unmount /data

# To mount again.. . check the /etc/fstab file what are the mounts
$ mount -a
